# Tests Directory

This directory contains test scripts and utilities for the project.

## planet_greeting.sh

A bash script that displays "Hello <random planet>" where the random planet is selected from the major and minor planets of our solar system.

### Usage

```bash
./planet_greeting.sh
```

### Planets included:
- Mercury
- Venus
- Earth
- Mars
- Jupiter
- Saturn
- Uranus
- Neptune
- Pluto
- Ceres
- Eris
- Makemake
- Haumea
- Gonggong
- Quaoar
- Varda
- Sedna
- Orcus
